## Installation
### Precompiled
  Download the *critical-(version).jar*, under the *Releases* tab  
  Place aformentioned jar in: /.minecraft/mods/

### Compile Source
  Extract the "critical-master.zip" archive and:
  
  On Windows:
  > gradlew build

  On Linux:
  > chmod +x ./gradlew  
  > ./gradlew build
